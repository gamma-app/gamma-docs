## Page options and layout

Per-page layout toggles in **Page options** → **Layout**:

- Page title  
- Page description  
- **Table of contents** (hide TOC on that page)  
- **Page outline** (H1/H2 headings)  
- Next/previous links  
- Page metadata  
- Tags  
- **Wide** layout (widens tables, cards, code blocks)

**Frontmatter (Git Sync):** `hidden: true` hides a page from the table of contents.
